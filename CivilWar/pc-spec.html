～「CivilWar」動作環境～

～「CivilWar」必要動作環境～

OS: Windows 7 64ビット版

CPU: Intel Core i5-6300 3.8GHz / AMD FX-4350 4.2GHz

RAM: 6GB


CPU: NVIDIA GeForce GT 640 / Radeon HD 7730

GPU RAM: 1GB

ストレージ: 700MB以上の空き容量


～「CivilWar」推奨動作環境～

「CivilWar」は必要環境を満たしたPCで快適にプレイすることができますが、滑らかな60FPSのフレームレートで最高のプレイ体験を楽しむためには推奨動作環境を用意する必要があります。※ゲーム内CGの関係により思うように動作しない場合があります。


OS: Windows 7 64ビット版

CPU: Intel i5-3570Tおよび同等品

RAM: 6GB


CPU: NVIDIA GeForce GT 640 / Radeon HD 7730

GPU RAM: 1GB

ストレージ: 700MB以上の空き容量
